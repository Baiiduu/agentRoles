# GitHub Docs

这个目录存放仓库协作与 GitHub 配置文档。

当前包含：

1. `github-collaboration-flow.md`
   从 Issue 到 Merge 的基础协作流程。
2. `review-ownership-boundaries.md`
   Review owner 的责任边界与 PR 路由规则。
3. `github-repository-settings-checklist.md`
   GitHub 网页端配置清单。
4. `ci-and-pre-commit.md`
   CI、pre-commit 与 required checks 说明。
5. `pre-push-cleanup-checklist.md`
   首次推送到 GitHub 前的本地产物清理清单。

维护建议：

1. 修改协作流程时，同步更新相关模板与说明文档。
2. 修改 GitHub 设置策略时，确保与 `.github/` 下的实际配置保持一致。
